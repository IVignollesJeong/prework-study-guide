GIT

-Git commands-

  • git status: checks what branch we are currently on
  • git checkout main or branch-name: moves current workspace to said branch
  • git checkout -b branch-name: creates a new branch (-b) and switches to it
  • code .: launches code in Visual Studio
  • git add -A: adds current code from VS to the current working branch
  • git commit -m "description of change": commits current code and adds a description message (-m)
  • git pull origin name-of-branch:
  • git push origin name-of-branch:
